# freeudemy
Installation guide


Step 1 : git clone https://github.com/Ajit-Tabiyad/freeudemy.git


Step 2: cd freeudemy  


Step 3 : pip3 install -r requirements.txt


Step 4 : python3 freeudemy.py




================== Thats it , now you will have free coupouns ==============
